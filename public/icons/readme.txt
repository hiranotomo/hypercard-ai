Classic Mac Icons

This directory should contain classic Mac-style icons for the stacks.
The icons shown in the screenshot are:

- Welcome: Classic Mac stack icon
- History: Scroll/document icon
- Adventure: Game controller icon
- Memories: Cloud/thought bubble icon
- Connect: Handshake icon
- Create: Pencil/pen icon
- About: Info icon
- Teleport: Rocket icon

These should be in .png or .svg format and styled to match
the classic Mac aesthetic (black and white, pixelated).
